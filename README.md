### 🏆 Flagship Masterpiece: PageHaven
**PageHaven** is a high-performance Digital Library platform that redefines how we interact with classical literature.

<p align="center">
  <img src="https://raw.githubusercontent.com/saniya1013/pagehaven-digital-library/main/landing_page_mockup_1777868841062.png" alt="PageHaven Mockup" width="100%" style="border-radius: 15px;"/>
</p>

#### **Advanced Specifications:**
- 🧠 **Cognitive Engine**: Powered by **Llama 3.1** via Groq API for sub-second AI response times.
- 🎨 **Visual Identity**: Custom **Blue-Cyan Design System** with Glassmorphism & 3D Tilt interactions.
- 🏗 **Architecture**: Fully decoupled RESTful architecture with a high-concurrency FastAPI core.
- 📊 **Data persistence**: Optimized MongoDB Atlas integration with advanced indexing for history tracking.
